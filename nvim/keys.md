# coc.nvim 配置总结

---

## 一、插入模式补全与编辑

| 键 | 功能 |
|---|------|
| `<Tab>` | • 补全菜单可见 → 确认当前选中项（上屏）<br>• 菜单不可见且光标前为空白 → 插入普通 `<Tab>`<br>• 菜单不可见且光标前有字符 → 手动触发补全 |
| `<S-Tab>` | • 补全菜单可见 → 选择上一项<br>• 菜单不可见 → 删除前一个字符（`<C-h>`） |
| `<CR>` | • 补全菜单可见 → 确认选中项<br>• 菜单不可见 → 换行并调用 `coc#on_enter()`（自动缩进/格式化） |
| `<C-space>`（Neovim）<br>`<C-@>`（Vim） | 手动触发补全（`coc#refresh()`） |

---

## 二、普通模式代码导航与诊断

| 键 | 功能 |
|---|------|
| `[g` | 跳转到上一个诊断（错误/警告） |
| `]g` | 跳转到下一个诊断 |
| `gd` | 跳转到定义 |
| `gy` | 跳转到类型定义 |
| `gi` | 跳转到实现 |
| `gr` | 查找所有引用 |
| `K` | 显示光标下符号的文档（hover） |
| `CursorHold` | 自动高亮光标下符号及其所有引用 |

## 三、代码操作与重构（`<leader>` 默认为 `\`）

| 键 | 功能 |
|---|------|
| `<leader>rn` | 重命名当前符号（全局） |
| `<leader>f` | 格式化选中的代码（可视模式）或当前行（普通模式） |
| `<leader>a` | 对选中区域执行代码动作（快速修复、提取等） |
| `<leader>ac` | 对光标所在位置执行代码动作 |
| `<leader>as` | 对整个缓冲区执行可用的源码动作 |
| `<leader>qf` | 自动修复当前行的诊断（优先选择 quickfix 动作） |
| `<leader>re` | 全局重构代码动作（普通模式） |
| `<leader>r` | 对选中区域执行重构代码动作（可视模式） |
| `<leader>cl` | 运行当前行的 CodeLens 动作（如运行测试） |

## 六、范围选择与列表操作

| 键 | 功能 |
|---|------|

| `<space>e` | 打开 CocList 管理扩展 |

| `<space>j` | 在 CocList 中执行下一项的默认动作 |
| `<space>k` | 在 CocList 中执行上一项的默认动作 |

## 七、新增 Ex 命令

| 命令 | 功能 |
|---|------|
| `:Format` | 格式化整个当前缓冲区 |
| `:Fold` | 根据语言服务器提供的折叠范围折叠代码 |

## 八、自动命令与状态行

| 配置项 | 作用 |
|---|------|
| `autocmd FileType typescript,json` | 对这些文件类型设置 `formatexpr` 通过 Coc 进行格式化 |
| `set statusline^=...` | 在状态栏左侧显示 coc.nvim 状态（如语言服务器名、错误数）及当前函数名 |

---

## 九、其他说明

- **`<leader>`** 默认键为 `\`，可自行修改 `mapleader`。
- **`<space>`** 作为前缀键用于 CocList 相关映射，不是 `<leader>` 的一部分。
- 补全菜单的**上下选择**依赖原生方向键（`<Up>` / `<Down>`），无需额外配置。
- 若需实现"上下箭头仅移动高亮，不自动填充"，需要在 `coc-settings.json` 中添加 `"suggest.noselect": true`。

# Vim 核心快捷键总结

## 一、快速跳转

### 行内跳转
- `0` / `^` — 行首（首个字符/非空字符）
- `$` — 行尾
- `f{char}` / `F{char}` — 向右/左跳到指定字符
- `t{char}` — 跳到字符前一位
- `;` / `,` — 重复/反向重复上次查找

### 跨行跳转
- `gg` / `G` — 文件首行/末行
- `{n}G` — 跳到第 n 行
- `H` / `M` / `L` — 屏幕顶/中/底行
- `Ctrl+u/d` — 上下滚动半屏
- `Ctrl+b/f` — 上下翻整屏
- `zz/zt/zb` — 当前行居顶/中/底

### 单词/搜索跳转
- `w/b/e/ge` — 跳词
- `%` — 匹配括号间跳转
- `/pattern` / `?pattern` — 搜索
- `n/N` — 下/上搜索结果
- `*` / `#` — 搜索光标下单词

## 二、剪切/复制/粘贴

### 剪切 (d/c)
- `dd` — 整行
- `dw/daw/diw` — 删单词（区别：aw含空格/iw不含）
- `d$` / `d0` — 到行尾/行首
- `dgg` / `dG` — 到文件头/尾
- `ci"` / `ci(` — 修改引号/括号内内容

### 复制 (y)
- `yy` — 整行
- `yw/yaw` — 复制单词
- `y$` / `yG` — 到行尾/文件尾

### 粘贴
- `p` — 光标后粘贴
- `P` — 光标前粘贴

## 三、编辑组合技
- `>>/<<` — 缩进/反缩进
- `g~aw/guw/gUw` — 翻转/小写/大写单词
- `.` — 重复上次修改
- `u` / `Ctrl+r` — 撤销/重做

## 四、典型工作流
1. 移动代码：`dd` → `p`
2. 复制替换：`yiw` → `viw` → `p`
3. 交换两行：`dd` → `k` → `P`

## 五、心法
- **文本对象**：`aw, iw, a", i(, a<` + `d/c/y`
- **少用方向键**，多用 `w b e ge`
- **`f/t` 精准瞄准**，**`;`/`,` 重复**
- **`.` 重复修改**
